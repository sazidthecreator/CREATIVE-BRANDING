# CREATIVE-BRANDING

Personal portfolio website for Sazid Hossain — Creative Technologist & AI Automation Strategist from Dhaka, Bangladesh.

Built with [Astro](https://astro.build), [React](https://react.dev), and [Tailwind CSS](https://tailwindcss.com).

## Tech Stack

- **Framework**: Astro 5
- **UI Components**: React 19
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/   # React components (Navbar, HeroSection, etc.)
├── layouts/      # Astro layout templates
├── pages/        # Astro pages (index, about, portfolio, etc.)
└── styles/       # Global CSS and design tokens
data/
└── projects.json # Portfolio project data
```
